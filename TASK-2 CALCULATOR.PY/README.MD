# Calculator Application

## Description

The Calculator Application is a Python-based command-line program that performs basic arithmetic operations on two numbers. The application provides a simple menu-driven interface that allows users to perform calculations, view previous calculations through a history feature, and exit the program safely.


## Features

* Addition
* Subtraction
* Multiplication
* Division
* Calculation History
* Error Handling for Invalid Inputs
* Division by Zero Protection
* Menu-Driven Interface
* Continuous Calculations Until Exit


## Technologies Used

* Python 3


## How to Run

1. Install Python 3 on your system.
2. Save the source code as `calculator.py`.
3. Open Terminal or Command Prompt.
4. Navigate to the project folder.
5. Run the following command:

```bash
python calculator.py
```



## Menu Options

```text
===== Calculator =====

1. Addition
2. Subtraction
3. Multiplication
4. Division
5. View History
6. Exit
```



## Usage

1. Select an operation from the menu.
2. Enter the first number.
3. Enter the second number.
4. View the calculated result.
5. Repeat calculations as needed.
6. View all previous calculations using the History option.
7. Exit the application when finished.
