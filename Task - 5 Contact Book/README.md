# Contact Book Application

## Description

The Contact Book Application is a Python-based command-line program designed to manage personal contacts efficiently. Users can add, view, search, update, and delete contact information. The application stores details such as name, phone number, email address, and address while also validating phone numbers and preventing duplicate entries.



## Features

* Add New Contacts
* View All Saved Contacts
* Search Contact by Name or Phone Number
* Update Existing Contact Details
* Delete Contact by Contact Number
* Store Name, Phone Number, Email, and Address
* Phone Number Validation (10 Digits Only)
* Duplicate Phone Number Prevention
* User-Friendly Menu Interface
* Input Validation and Error Handling



## Technologies Used

* Python 3


## How to Run

1. Install Python 3 on your system.
2. Save the program as `contact_book.py`.
3. Open Terminal or Command Prompt.
4. Navigate to the project folder.
5. Run the following command:

```bash
python contact_book.py
```

## Usage

### Add Contact

* Enter Name
* Enter Phone Number
* Enter Email Address
* Enter Address

### View Contacts

* Displays all saved contacts.

### Search Contact

* Search by Name
* Search by Phone Number

### Update Contact

* Update Name
* Update Phone Number
* Update Email Address
* Update Address

### Delete Contact

* Delete a contact using its contact number from the contact list.
