# To-Do List Application

## Description

This is a simple To-Do List Application developed using Python. The application helps users manage their daily tasks by allowing them to add, view, and remove tasks from a task list.

## Features

* Add new tasks
* View all tasks
* Remove completed or unwanted tasks
* Simple menu-driven interface
* Easy to use and beginner-friendly

## Technologies Used

* Python 3

## How to Run

1. Install Python on your system.
2. Save the program as `todo.py`.
3. Open Terminal or Command Prompt.
4. Navigate to the project folder.
5. Run the following command:

```bash
python todo.py
```

## Menu Options

1. Add Task
2. View Tasks
3. Remove Task
4. Exit
